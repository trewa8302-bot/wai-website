Mobile Accessibility Task Force Meeting Resolutions

Minutes | Topics | Resolutions

Resolutions reflect general consensus of participants present in a meeting. They are not formal decisions of the group. Refer to the Decisions page for formal decisions. Context for each resolution is available via its link into the minutes.

2017

  • 23 February: Mobile Accessibility Task Force Teleconference
    • touchscreen - have no additional techniques for 3.3.2, just add an example to the understanding documents, write the example that Patrick just did on the fly for G 131

2016

  • 17 November: Mobile Accessibility Task Force Teleconference
    • M11 - Accept changed language for success criteria, submit to working group with a note to email thread about SC 3.3.2 and 1.3.1 for working group evaluation
    • M9 - submit M9 as modified
    • M 16 - submit M16 as modified
  • 27 October: Mobile Accessibility Task Force Teleconference
    • M13 Orientation - this is been reviewed by task force and is now ready for working group review
    • focus trap - drop M7 No Focus Trap for 2.1
  • 20 October: Mobile Accessibility Task Force Teleconference
    • M6 - Patrick to make final edits and ready for working group
  • 29 September: Mobile Accessibility Task Force Teleconference
    • will accept M15 Programmatic Notifications and submit to working group
  • 8 September: Mobile Accessibility Task Force Teleconference
    • Accidental Activation (finalize) - https://www.w3.org/WAI/GL/mobile-a11y-tf/wiki/Proposed_revision_of_2.5.3 - remove first SC, beef up second and third SC
    • Accidental Activation (finalize) - https://www.w3.org/WAI/GL/mobile-a11y-tf/wiki/Proposed_revision_of_2.5.3 - remove first SC, make further editorial changes to 2nd/3rd SC, send 3rd to COGA for sync
  • 1 September: Mobile Accessibility Task Force Teleconference
    • target size is 48 and 24 px
    • SC Target Size is accepted
  • 18 August: Mobile Accessibility Task Force Teleconference
    • SC Pointer with Assistive Technology - https://www.w3.org/WAI/GL/mobile-a11y-tf/wiki/Proposed_SC_Pointer_with_Assistive_Technology#SC_Shortname - https://www.w3.org/WAI/GL/mobile-a11y-tf/wiki/Proposed_SC_Pointer_with_Assistive_Technology#Benefits ready for WCAG
  • 23 June: Mobile Accessibility Task Force Teleconference
    • feedback from WCAG working group - add pointer definition to glossary and reference this definition where we generally mean pointing device
  • 9 June: Mobile Accessibility Task Force Teleconference
    • Finish 2.6.1 https://www.w3.org/WAI/GL/mobile-a11y-tf/wiki/Guideline_2.6:_Make_it_easier_to_use_the_physical_features_of_the_phone. - further discussion needed to write clear definition of pressure-sensitive input, to come up with use cases (easy and hard ones). see survey that will come on mailing list
  • 2 June: Mobile Accessibility Task Force Teleconference
    • remove force touch and touch from definition of device manipulation
  • 21 April: Mobile Accessibility Task Force Teleconference
    • review changes to Touch and Pointer - Accept the proposal as amended above (in wiki) for submission to WCAG WG.
  • 3 March: Mobile Accessibility Task Force Teleconference
    • 2.5.2 proposed Understanding text https://w3c.github.io/Mobile-A11y-Extension/#swipe-trap - revised Understanding text and to the Success Criteria of 2.5.2 No Swipe Trap to No Touch Trap
  • 18 February: Mobile Accessibility Task Force Teleconference
    • Finish 2.5.3 - https://w3c.github.io/Mobile-A11y-Extension/#touch-and-pointer - accept Intent for 2.5.2
    • Finish 2.5.3 - https://w3c.github.io/Mobile-A11y-Extension/#touch-and-pointer - accept Understanding for 2.5.2
  • 11 February: Mobile Accessibility Task Force Teleconference
    • Revisions to 2.5.3 Failure from Jonathan https://lists.w3.org/Archives/Public/public-mobile-a11y-tf/2016Feb/0008.html - Accept updates to the FM001 Failure of 2.5.3
